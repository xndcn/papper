## 1. 游戏概述与背景故事

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 纸翼传说 (Paper Wings Legend) |
| 游戏类型 | 2D 像素风 Roguelike 纸飞机竞速 |
| 目标平台 | Web (浏览器)，未来扩展至移动端 |
| 单局时长 | 10-15 分钟（一次锦标赛 Run） |
| 核心受众 | 喜欢 Roguelike 和休闲竞速的玩家，年龄 12+ |

### 1.2 背景故事

在一个阳光明媚的小镇上，有一个叫小翼的孩子。他不擅长运动，成绩也不算拔尖，但他有一双灵巧的手和一颗永不停歇的心——他热爱折纸飞机。

课间十分钟，当别的孩子在操场上追逐打闹时，小翼总是蹲在走廊的角落，用一张张从作业本上撕下的纸，折出一架又一架纸飞机。他看着自己的飞机划过走廊、飞越栏杆、在风中旋转，心中涌起一种难以言喻的自豪。

某天，小翼在公园里偶然目睹了一场纸飞机比赛。选手们投掷的纸飞机在空中翻转、滑翔、冲刺，姿态优美得令人窒息。他看到冠军高举奖杯的那一刻，心中燃起了一团火——他也要成为纸飞机冠军。

从班级的小小对决，到校内选拔赛，到区赛、市赛，一路披荆斩棘，最终站上全国大赛的舞台。这是一段关于梦想、坚持和成长的旅程，也是一段纸飞机从稚嫩走向传说的故事。

### 1.3 设计调性

- **温暖**：故事基调是积极向上的，NPC 对话充满关怀与鼓励
- **励志**：失败不是终点而是成长的养料，每一次跌倒都能学到新东西
- **童趣**：像素风美术、手绘风 UI、欢快的音乐，唤起童年折纸飞机的记忆
- **深度**：在温馨外表下藏着丰富的策略层，飞机配置、技能选择、天气应对都需要动脑思考

### 1.4 设计核心理念

> "每一次折叠都是一次尝试，每一次飞行都是一次冒险。"

纸飞机天生就带有"不确定性"——你永远不知道它会飞多远、飞多高。这种不确定性恰好与 Roguelike 的核心精神契合。我们要做的不是消除不确定性，而是给玩家足够的工具去驾驭它。

## 2. 核心游戏循环

纸翼传说采用**混合模式设计**，将持久化的大地图探索与 Roguelike 式的锦标赛 Run 有机结合。

### 2.1 大地图探索（持久化，Pokemon 式）

大地图是玩家的"家"，所有进度在这里永久保留。采用**节点地图**设计，不是自由行走，而是在固定场景节点之间移动。

#### 场景节点一览

| 节点 | 功能 | 解锁条件 |
|------|------|----------|
| **家** | 存档、休息、恢复状态、查看图鉴 | 初始可用 |
| **学校** | 学习新折纸技巧、解锁新机型、升级永久技能树 | 初始可用 |
| **后山** | 练习场，试飞新飞机、测试配置、熟悉操作 | 初始可用 |
| **小卖部** | 购买零件、消耗品、特殊材料 | 初始可用 |
| **公园** | 与 NPC 交流、接受挑战、获取情报 | 完成第一场班级赛后解锁 |
| **比赛场地** | 进入锦标赛 Run（不同级别） | 达到对应声望等级后解锁 |
| **废弃仓库** | 稀有零件交易、隐藏 NPC | 累计失败 5 次后出现 |
| **山顶展望台** | 全国赛前的特殊训练区 | 击败第 4 位馆主后解锁 |

#### 节点交互设计

- 每个节点有 1-3 个可交互的 NPC
- NPC 对话随全局进度更新（详见[叙事系统](#7-叙事系统)）
- 部分节点有隐藏事件，需要特定条件触发（如带着特定飞机访问后山）
- 节点之间用像素风的小镇地图连线，玩家点击节点即可移动

#### 大地图经济

| 货币 | 获取途径 | 用途 |
|------|----------|------|
| **金币** | 比赛奖励、NPC任务、日常签到 | 在小卖部购买零件和消耗品 |
| **经验** | 每次 Run 结算（无论胜败） | Meta 进度升级 |
| **声望** | 赢得锦标赛、击败馆主 | 解锁更高级别的比赛场地 |
| **稀有纸张** | 隐藏事件、成就奖励 | 进化飞机的必要材料 |

### 2.2 锦标赛 Run（Roguelike 循环）

一次 Run 代表一场锦标赛，是游戏的核心战斗循环。

#### Run 基本结构

- 每次 Run 包含 **5-7 场比赛**
- 整个 Run 预计时长 **10-15 分钟**
- 采用**分支路径地图**（类似 Slay the Spire）
- 玩家在每个分叉点选择下一个节点，不同路径意味着不同的风险与回报

#### 路径节点类型

| 节点类型 | 图标 | 描述 | 出现频率 |
|----------|------|------|----------|
| **普通比赛** | 纸飞机图标 | 标准 1v1 或 1v3 竞速 | 高 |
| **精英对手** | 火焰纸飞机 | 更强的对手，更好的奖励 | 中 |
| **商店** | 小卖部图标 | 用金币购买零件或 buff | 中 |
| **休息站** | 长椅图标 | 恢复耐久 / 升级一个零件 | 低 |
| **随机事件** | 问号图标 | 风险/回报选择事件 | 中 |
| **Boss（馆主）** | 皇冠图标 | 每次 Run 的最终挑战 | 固定（终点） |

#### 随机事件示例

- **捡到旧报纸**：获得一个随机普通零件，或用它交换一个未鉴定的零件（可能是稀有）
- **路边的老爷爷**：教你一个飞行技巧（获得临时被动 buff），但下一场比赛逆风概率增加
- **迷路的小猫**：花时间帮助它（失去一个零件槽位直到 Run 结束），但获得"好运"加成（稀有零件掉落率 +30%）
- **突降暴雨**：选择等待（跳过下一场比赛的奖励）或冒雨前进（飞机耐久 -20%，但获得暴风天气抗性 buff）

#### Run 内资源管理

- **飞机耐久**：每场比赛后根据碰撞和天气损耗，耐久归零则飞机性能大幅下降
- **技能冷却**：部分强力技能有跨比赛的冷却
- **零件背包**：Run 中最多携带 8 个备用零件，需要在取舍间做出策略选择
- **临时金币**：Run 中获得的金币可在 Run 内商店使用，Run 结束后部分转化为永久金币

#### Run 结算

| 结算项 | 胜利 | 失败 |
|--------|------|------|
| 永久经验 | 基础经验 + 奖励经验 | 基础经验（不为零） |
| 金币 | 全额保留 | 保留 50% |
| 零件 | 可保留 Run 中获得的零件（数量有限） | 可保留 1 个 |
| 声望 | +声望值 | 不变 |
| 剧情进度 | 推进 | 可能触发失败专属剧情 |

### 2.3 单场比赛流程

每场比赛是一个完整的"准备-发射-飞行-结算"循环，时长约 **90 秒**。

#### 阶段一：赛前信息（5 秒）

展示本场比赛的关键信息，让玩家做出策略判断：

- **赛道条件**：天气（晴/风/雨/暴风）、风向（顺/逆/侧）、赛道距离（短/中/长）
- **对手信息**：对手飞机类型、性格倾向、历史交手记录
- **特殊规则**：部分比赛有特殊规则（如"不能使用加速技能"、"必须经过指定检查点"）

#### 阶段二：构建阶段（15-20 秒）

玩家在此阶段调整飞机配置以应对本场比赛条件：

- **更换零件**：从背包中替换当前装备的零件
- **选择主动技能**：从已解锁的技能中装备 2-3 个
- **调整发射参数**：设定发射角度（0-90 度）和力度（10%-100%）
- **查看属性预览**：实时显示当前配置的五维属性雷达图

> 设计意图：构建阶段是策略深度的核心来源。不同天气、不同对手需要不同的配置方案。短赛道适合速度型高力度低角度发射，长赛道适合滑翔型中等角度中等力度发射。

#### 阶段三：发射（3 秒）

- 玩家确认角度和力度后点击"发射"按钮
- 播放发射动画（像素风的手臂投掷动作）
- 飞机进入飞行阶段

#### 阶段四：飞行阶段（60-70 秒）

飞机发射后**自动飞行**，基于物理模拟系统计算飞行轨迹。玩家的操作集中在**主动技能释放**上：

- 飞行画面为横版卷轴视角，摄像机跟随玩家飞机
- 同屏可见对手飞机的相对位置
- 玩家通过点击/快捷键释放装备的主动技能
- 飞行过程中可能触发被动 buff 的三选一选择界面
- 赛道上有随机出现的气流区域（助力/阻力）和障碍物

**飞行物理模型关键参数：**

| 参数 | 影响因素 | 说明 |
|------|----------|------|
| 水平速度 | Speed 属性 + 风向 + 技能加成 | 决定前进速度 |
| 垂直速度 | Glide 属性 + 发射角度 + 气流 | 决定高度变化 |
| 偏航角 | Stability 属性 + 侧风强度 | 决定是否偏离航线 |
| 碰撞衰减 | Durability 属性 + 碰撞类型 | 碰撞后速度损失比例 |
| 特技执行 | Trick 属性 + 技能等级 | 花式动作成功率和效果 |

#### 阶段五：结算（5-10 秒）

- 显示本场比赛排名（1v1 或 1v3）
- 展示飞行统计：飞行距离、最高速度、技能使用次数、花式动作得分
- 发放奖励：新零件（随机掉落）、金币、经验
- 若是精英/Boss 赛，额外展示特殊奖励

## 3. 纸飞机系统

纸飞机是游戏的核心资产，玩家围绕飞机的收集、配置和进化展开长期游玩。

### 3.1 基础机型分类（三大类）

游戏中的所有纸飞机归属于三大类别，每个类别有鲜明的飞行特性和策略定位。

#### 速度型（飞镖系）

| 属性 | 评级 |
|------|------|
| Speed | ★★★★★ |
| Glide | ★★☆☆☆ |
| Stability | ★★☆☆☆ |
| Trick | ★☆☆☆☆ |
| Durability | ★★★☆☆ |

- **飞行特征**：直线飞行速度极快，初始冲刺力强
- **优势**：短赛道和顺风天气的王者，能迅速拉开距离
- **劣势**：侧风和暴风天气表现极差，滑翔距离短
- **适合场景**：短距离赛道、顺风条件、需要爆发力的比赛
- **代表机型**：折纸飞镖、银箭、闪电飞镖

#### 特技型（翅膀系）

| 属性 | 评级 |
|------|------|
| Speed | ★★★☆☆ |
| Glide | ★★★☆☆ |
| Stability | ★★☆☆☆ |
| Trick | ★★★★★ |
| Durability | ★★☆☆☆ |

- **飞行特征**：可执行翻转、旋转等特殊动作，飞行姿态多变
- **优势**：花式操作可获得额外加速和得分，操控类技能效果增强
- **劣势**：基础飞行稳定性不高，操作要求高
- **适合场景**：障碍物多的赛道、侧风（可利用做花式）、需要灵活操作的比赛
- **代表机型**：蝴蝶翼、风舞者、凤凰纸鸢

#### 稳定型（滑翔系）

| 属性 | 评级 |
|------|------|
| Speed | ★★☆☆☆ |
| Glide | ★★★★★ |
| Stability | ★★★★★ |
| Trick | ★★☆☆☆ |
| Durability | ★★★☆☆ |

- **飞行特征**：平稳滑翔，受风影响极小，滞空时间长
- **优势**：长赛道和恶劣天气的首选，飞行路线稳定可预测
- **劣势**：速度慢，短赛道容易被速度型碾压
- **适合场景**：长距离赛道、逆风/暴风条件、需要稳定发挥的比赛
- **代表机型**：经典滑翔机、白鸥、云端旅者

### 3.2 机型五维属性

每架纸飞机由五个核心属性定义，属性值范围为 **1-100**。

| 属性 | 英文 | 作用 | 公式影响 |
|------|------|------|----------|
| **速度** | Speed | 飞行水平速度的基础值 | 水平速度 = Speed × 风向系数 × buff 系数 |
| **滑翔** | Glide | 滑翔距离和滞空时间 | 下落速率 = 基础重力 / (Glide × 0.1) |
| **稳定** | Stability | 抗风能力和飞行路线偏差 | 偏航角 = 侧风力度 / (Stability × 0.15) |
| **特技** | Trick | 花式动作的可用种类和效果 | 花式成功率 = Trick × 技能等级系数 |
| **耐久** | Durability | 碰撞承受力和恶劣天气抗性 | 碰撞速度损失 = 基础损失 × (1 - Durability × 0.008) |

**属性上限与来源：**

- 基础属性：由机型决定，范围 10-60
- 零件加成：由装备的零件提供，范围 0-30（每个零件）
- 进化加成：由机型进化等级决定，范围 0-15
- 永久技能树加成：由 Meta 进度提供，范围 0-10
- 单项属性硬上限：**100**

### 3.3 模块化零件系统

每架飞机有 **5 个零件槽位**，不同机型的可用槽位数量不同。

#### 槽位定义

| 槽位 | 英文 | 主要属性影响 | 可选特殊效果 |
|------|------|-------------|-------------|
| **机鼻** | Nose | Speed, Durability | 穿透障碍、破风减阻 |
| **机翼** | Wing | Glide, Stability | 展翼滑翔、侧风利用 |
| **尾翼** | Tail | Stability, Trick | 急转弯增强、尾流控制 |
| **涂装** | Coating | 全属性小幅加成 | 天气抗性、视觉效果 |
| **配重** | Weight | Speed, Glide（反向关系） | 调整重心、改变飞行弧线 |

#### 机型可用槽位

| 机型类别 | 初始槽位数 | 可用槽位 | 进化后额外槽位 |
|----------|-----------|----------|---------------|
| 速度型 | 3 | Nose, Coating, Weight | +Wing 或 +Tail |
| 特技型 | 3 | Wing, Tail, Coating | +Nose 或 +Weight |
| 稳定型 | 3 | Wing, Tail, Weight | +Nose 或 +Coating |

### 3.4 零件稀有度

#### 普通（白色边框）

- 属性加成：+3 ~ +8（单属性）
- 特殊效果：无
- 获取途径：比赛基础奖励、小卖部购买
- 掉落率：70%

#### 稀有（蓝色边框）

- 属性加成：+6 ~ +15（单属性或双属性）
- 特殊效果：50% 概率附带一个特殊效果
- 获取途径：精英比赛奖励、商店购买、随机事件
- 掉落率：25%

**稀有零件特殊效果示例：**

| 零件名 | 槽位 | 属性 | 特殊效果 |
|--------|------|------|----------|
| 锋锐鼻锥 | Nose | Speed +12 | 顺风时额外加速 10% |
| 广展翼面 | Wing | Glide +10, Stability +5 | 高度低于 30% 时自动获得微弱上升气流 |
| 铅笔配重 | Weight | Speed +8, Glide -3 | 发射力度超过 80% 时，初始冲刺距离 +20% |

#### 传说（金色边框）

- 属性加成：+12 ~ +25（双属性或三属性）
- 特殊效果：100% 附带一个独特被动效果
- 获取途径：击败馆主、隐藏事件、成就奖励
- 掉落率：5%

**传说零件示例：**

| 零件名 | 槽位 | 属性 | 独特被动 |
|--------|------|------|----------|
| 龙息鼻锥 | Nose | Speed +20, Durability +10 | "龙息"：每 15 秒自动释放一次短距离冲刺 |
| 凤羽之翼 | Wing | Glide +18, Trick +12 | "涅槃"：耐久归零时恢复 30% 耐久（每 Run 限一次） |
| 星辰涂装 | Coating | 全属性 +5 | "星光指引"：夜间赛道全属性额外 +8 |
| 陨铁配重 | Weight | Speed +15, Glide -5 | "坠星"：从高处俯冲时速度加成翻倍 |

### 3.5 套装效果

收集同系列零件可以激活额外的套装加成，鼓励玩家围绕特定策略搭配零件。

#### 套装列表

| 套装名 | 所需件数 | 3 件效果 | 5 件效果 |
|--------|----------|----------|----------|
| **空气动力套装** | 5 | Glide +10, Stability +10 | 额外：滑翔距离 +25%，受风影响 -30% |
| **暴风骑士套装** | 5 | Speed +10, Stability +8 | 额外：逆风时不减速反加速 10%，暴风天气伤害免疫 |
| **花式大师套装** | 5 | Trick +12, Durability +6 | 额外：花式动作必定成功，花式得分 +50% |
| **纸甲战士套装** | 5 | Durability +15, Speed +5 | 额外：碰撞完全免疫，但 Trick -10 |
| **风行者套装** | 3 | Speed +8, Glide +5 | — （仅 3 件套） |
| **幸运星套装** | 3 | 稀有零件掉落率 +15% | — （仅 3 件套） |

#### 套装件数与机型槽位的策略关系

由于不同机型可用槽位不同，5 件套装需要在进化后获得额外槽位才能完全激活。这创造了一个有趣的策略决策：

- **3 件套**：任何机型都能凑齐，是通用选择
- **5 件套**：需要进化后的高级机型，是后期追求目标
- 部分零件可以跨套装使用，但同时只能计入一个套装

### 3.6 进化路径

基础机型可以通过达成特定条件进化为更高级的形态。进化是不可逆的永久升级。

#### 进化条件与路径

**速度型进化路径：**

```
折纸飞镖 (Lv.1)
├── 银箭 (Lv.2) ── 条件：Speed 属性达到 50 + 赢得 10 场比赛
│   └── 闪电飞镖 (Lv.3) ── 条件：在顺风中达到最高速度纪录 + 击败馆主"疾风少年"
└── 穿云锥 (Lv.2) ── 条件：Durability 属性达到 40 + 累计穿越 50 个障碍物
    └── 破晓之矛 (Lv.3) ── 条件：在暴风天气中赢得 5 场比赛 + 1 张稀有纸张
```

**特技型进化路径：**

```
蝴蝶翼 (Lv.1)
├── 风舞者 (Lv.2) ── 条件：Trick 属性达到 50 + 完成 30 次花式动作
│   └── 凤凰纸鸢 (Lv.3) ── 条件：在单场比赛中完成 5 次花式 + 击败馆主"花式女王"
└── 螺旋翼 (Lv.2) ── 条件：Stability 属性达到 40 + 利用侧风完成 20 次花式
    └── 龙卷风行者 (Lv.3) ── 条件：在侧风中连续赢得 3 场比赛 + 1 张稀有纸张
```

**稳定型进化路径：**

```
经典滑翔机 (Lv.1)
├── 白鸥 (Lv.2) ── 条件：Glide 属性达到 50 + 累计滑翔距离超过 10000 米
│   └── 云端旅者 (Lv.3) ── 条件：单次飞行滑翔距离超过 500 米 + 击败馆主"滑翔老人"
└── 堡垒纸盾 (Lv.2) ── 条件：Durability 属性达到 50 + 累计碰撞后存活 100 次
    └── 不沉之翼 (Lv.3) ── 条件：在暴风天气中不受任何碰撞完成比赛 + 1 张稀有纸张
```

#### 进化收益

| 进化等级 | 基础属性提升 | 槽位变化 | 外观变化 |
|----------|------------|---------|---------|
| Lv.1 → Lv.2 | 全属性 +5 | +1 新槽位 | 线条更精致，颜色微调 |
| Lv.2 → Lv.3 | 全属性 +10，主属性额外 +5 | 解锁隐藏槽位效果 | 全新精灵图，带光效/拖尾 |

## 4. 技能系统

技能系统为飞行阶段提供主动操作空间，让"自动飞行"不再只是旁观。

### 4.1 主动技能

主动技能在**赛前构建阶段装备**，在**飞行阶段点击触发**。每架飞机可装备 **2-3 个主动技能**（取决于机型和进化等级），每个技能有独立的冷却时间。

#### 主动技能列表

| 技能名 | 图标 | 效果 | 冷却 | 持续 | 适配机型 |
|--------|------|------|------|------|----------|
| **加速冲刺** | 火焰箭头 | 速度提升 80%，但稳定性降低 30% | 12 秒 | 2 秒 | 速度型增强 |
| **急转弯** | 弯曲箭头 | 立即改变飞行方向 45 度 | 8 秒 | 瞬发 | 特技型增强 |
| **护盾** | 蓝色光圈 | 免疫一次碰撞和逆风效果 | 18 秒 | 3 秒 | 稳定型增强 |
| **上升气流** | 向上螺旋 | 利用热气流获得高度，上升 30% 高度 | 15 秒 | 2 秒 | 滑翔系增强 |
| **旋风突进** | 旋转图标 | 旋转穿过障碍物，同时小幅加速 | 10 秒 | 1.5 秒 | 特技型增强 |
| **尾流牵引** | 虚线跟随 | 锁定前方最近对手，进入其尾流（减阻加速） | 20 秒 | 4 秒 | 通用 |
| **纸弹射击** | 小纸团 | 向前方发射纸弹，命中对手使其减速 15% | 14 秒 | 瞬发 | 速度型增强 |
| **折叠变形** | 变形图标 | 临时切换飞机形态（速度型↔稳定型），持续期间获得另一类型 50% 属性 | 25 秒 | 5 秒 | 特技型增强 |

#### 技能升级

- 每个技能有 3 个等级
- 升级途径：在学校使用经验升级
- 升级效果：冷却缩短、效果增强、持续时间延长

| 等级 | 冷却缩减 | 效果增幅 | 解锁条件 |
|------|----------|---------|---------|
| Lv.1 | 基础值 | 基础值 | 默认 |
| Lv.2 | -15% | +20% | 使用该技能 30 次 |
| Lv.3 | -25% | +40% | 使用该技能 100 次 + 消耗经验 |

### 4.2 被动 Buff

被动 buff 在**飞行阶段随机触发**，采用 **三选一** 机制（类似 Hades 的祝福选择）。

#### 触发条件

| 触发事件 | 出现概率 | 说明 |
|----------|---------|------|
| 超越对手 | 60% | 每次超越一个对手时可能触发 |
| 到达检查点 | 40% | 通过赛道中的检查点时可能触发 |
| 完成花式动作 | 80% | 成功执行花式动作后高概率触发 |
| 飞行高度达到峰值 | 30% | 达到当次飞行最高点时可能触发 |

| **逆风之心** | 天气 | 逆风减速效果降低 50% | 稀有 |

| **精准投掷** | 稳定 | Stability 临时 +20 | 普通 |

### 4.3 技能与飞机类型协同

不同类型的飞机对不同类别的技能有天然的增幅效果，鼓励玩家围绕机型特点构建技能搭配。

| 飞机类型 | 增幅类别 | 增幅效果 | 设计意图 |
|----------|---------|---------|---------|
| **速度型** | 加速类技能（加速冲刺、尾流牵引） | 效果 +30%，冷却 -20% | 强化速度型的爆发优势 |
| **特技型** | 操控类技能（急转弯、旋风突进、折叠变形） | 效果 +30%，冷却 -20% | 强化特技型的操作上限 |
| **稳定型** | 防御类技能（护盾、上升气流） | 效果 +30%，冷却 -20% | 强化稳定型的生存能力 |

> 设计意图：协同系统不是强制绑定，而是提供激励。速度型飞机也可以装防御技能来弥补短板，只是不会获得增幅加成。玩家可以选择"扬长"或"补短"，两种策略都是可行的。

## 5. 属性克制与天气系统

### 5.1 三角克制体系

三大飞机类型之间存在循环克制关系，为策略选择增添深度。

```
      速度型
     /      \
   克制    被克制
   /          \
特技型 ←——克制—— 稳定型
```

| 克制关系 | 原因说明 | 属性优势幅度 |
|----------|---------|-------------|
| **速度型 > 特技型** | 高速飞行让花式操作来不及施展，速度碾压技巧 | 约 15-20% 属性优势 |
| **特技型 > 稳定型** | 灵活的花式操作可以利用气流和地形破解稳定型的优势 | 约 15-20% 属性优势 |
| **稳定型 > 速度型** | 稳定的飞行路线在长距离中逐渐消耗速度型的优势，后程反超 | 约 15-20% 属性优势 |

#### 克制机制实现

克制关系通过以下方式体现：

- **直接属性修正**：对阵被克制类型时，主属性 +10%
- **赛道生成偏向**：当玩家面对克制自己的对手时，赛道条件会稍有利于玩家（降低挫败感）
- **被动触发率**：对阵被克制类型时，有利 buff 触发概率 +10%

> 设计意图：15-20% 的克制优势足以影响比赛结果，但不是决定性的。技能选择、零件配置和操作水平仍能翻盘。

### 5.2 天气系统

每场比赛随机生成天气条件，不同天气对三大类型飞机的影响截然不同。

#### 天气效果矩阵

| 天气 | 速度型影响 | 特技型影响 | 稳定型影响 | 出现概率 |
|------|-----------|-----------|-----------|---------|
| **顺风** | Speed +25%（大幅加速） | Speed +15%（中等加速） | Speed +8%（小幅加速） | 20% |
| **逆风** | Speed -25%（大幅减速） | Speed -15%（中等减速） | Speed -8%（小幅减速） | 20% |
| **侧风** | 偏航角 +60%（偏航严重） | 可利用做花式（Trick +15） | 偏航角 +5%（几乎不受影响） | 20% |
| **暴风** | 全属性波动 ±30%（极其不稳定） | 全属性波动 ±15%（不稳定） | 全属性波动 ±5%（相对稳定） | 10% |
| **温润（无风）** | 正常表现 | Trick -10（花式受限，无风可借） | 正常表现 | 30% |

#### 天气的策略意义

天气系统的核心目的是让玩家**不能只练一种飞机**：

- 如果全程使用速度型，遇到逆风/暴风就会非常痛苦
- 如果全程使用稳定型，顺风天气就无法最大化收益
- 最优策略是**准备多种飞机并根据天气灵活切换**

#### 天气预报机制

- 进入 Run 时，可以看到**前 2 场比赛的天气**
- 第 3 场及之后的天气只显示"可能的天气范围"
- 部分 NPC 情报（公园伙伴）可以提前透露后续天气
- 休息站可以花费金币"查看下一场天气"

#### 复合天气（后期出现）

在市赛和全国赛阶段，会出现复合天气条件：

| 复合天气 | 组合 | 特殊效果 |
|----------|------|---------|
| **雷暴** | 暴风 + 逆风 | 随机闪电击中飞机（Speed 瞬间归零 0.5 秒） |
| **台风眼** | 暴风 → 无风 → 暴风 | 赛道中段有一片无风区域，前后是暴风 |
| **热浪上升流** | 温润 + 特殊气流 | 赛道底部有热气流区域，利用得当可大幅提升高度 |
| **迷雾** | 温润 + 视野限制 | 视野缩小 50%，对手位置不可见 |

## 6. 对手与 NPC 系统

### 6.1 对手性格原型

游戏中的对手不是简单的数值堆砌，而是有鲜明性格的角色。每个对手的行为模式由其性格原型决定。

### 7.2 Hades 式渐进叙事

借鉴 Hades 的叙事系统，每次 Run 结束后（无论胜败）回到大地图时，NPC 都有新的对话内容。

#### 对话触发机制

```
对话选择逻辑：
1. 检查是否有特定事件触发的对话（最高优先级）
2. 检查当前全局进度是否有新的进度对话
3. 检查失败次数是否触发安慰/鼓励对话
4. 从通用对话池中随机选择（最低优先级）
```

#### 对话池分类

| 类别 | 触发条件 | 示例 |
|------|---------|------|
| **进度对话** | 全局进度达到特定阶段 | 妈妈在主角到达区赛后："我小时候也会折纸飞机呢，不过都是歪歪扭扭的。" |
| **事件对话** | 完成特定事件 | 林小冲在被击败后的第一次在公园相遇："嘿！我可不是来复仇的……好吧，也许是。再来一场怎么样？" |
| **失败对话** | 连续失败达到特定次数 | 爸爸在连续失败 3 次后："有时候飞机飞不远，不是飞机的问题，是风的问题。等风变了，你的飞机会飞得很远。" |
| **随机对话** | 无特定触发条件 | 小卖部张叔："今天进了一批新的彩色纸，你要不要来看看？" |
| **隐藏对话** | 罕见条件触发 | 带着全传说零件飞机访问废品站大姐："小鬼头，你这飞机……不对，让我仔细看看。这是……！" |

#### 对手关系发展

馆主和关键对手在多次交手后会逐渐展现更多性格层面：

| 阶段 | 对白特征 | 关系变化 |
|------|---------|---------|
| **初次相遇** | 挑衅/好奇 | 陌生人 |
| **首次战败** | 不服/惊讶 | 对手 |
| **多次交手** | 认可/尊重 | 竞争者 |
| **后期互动** | 友善/鼓励 | 朋友 |

### 7.3 失败叙事

失败是纸翼传说叙事体验的重要组成部分，而不是需要回避的负面体验。

#### 失败不惩罚原则

- **不丢失大地图进度**：解锁的节点、学会的折法、进化的飞机都永久保留
- **获得经验（Meta 货币）**：每次 Run 无论胜败都获得经验，失败获得的经验约为胜利的 60-80%
- **可保留一个零件**：即使 Run 失败，也可以从本次 Run 获得的零件中带走 1 个

#### 失败专属内容

| 内容类型 | 触发条件 | 示例 |
|----------|---------|------|
| **安慰对话** | 每次失败后回家 | 妈妈："累了吧？先吃口饭，飞机明天再折。" |
| **成长对话** | 累计失败 5/10/20 次 | 爸爸：讲述自己年轻时在市赛失败的故事，给予飞行技巧提示 |
| **隐藏NPC** | 累计失败 5 次 | 废弃仓库出现，废品站大姐提供特殊零件交易 |
| **支线剧情** | 对特定馆主失败 3 次 | 该馆主主动来找主角，私下传授一招克制自己的技巧 |
| **隐藏成就** | 特定失败条件 | "永不放弃"成就：连续失败 10 次后仍然进入 Run |

#### 馆主失败专属剧情

每位馆主在首次击败玩家后都有特殊表现：

- **林小冲**：嘲笑但会给一个速度型零件，"这个送你，下次再来的时候别飞这么慢"
- **苏雅**：冷淡地留下一句"回去多练练"，但偷偷在主角背包里放了一个花式教程
- **王大山**：沉默地拍了拍主角的肩膀，然后离开
- **赵小星**：分析主角失败的原因并给出建议（提供天气预报提示功能）
- **陈老**：微笑着说"再来吧，我等着你"

## 9. 纸飞机折法图鉴

### 9.1 像素风动画教学

解锁新飞机时，播放一段精心制作的像素风折纸步骤动画。

#### 动画规格

| 参数 | 规格 |
|------|------|
| 分辨率 | 128×128 像素（放大显示） |
| 帧率 | 8 FPS（像素风标准） |
| 每步时长 | 2-3 秒 |
| 总步骤 | 简单机型 4-6 步，复杂机型 8-12 步 |
| 配乐 | 轻快的折纸音效（纸张翻折声） |

#### 动画流程

1. **展示白纸**：一张方形/长方形白纸出现在画面中央
2. **逐步折叠**：精灵图逐帧展示每个折叠动作
   - 折叠线以虚线预先标记
   - 折叠方向以箭头动画指示
   - 折叠完成后短暂停留展示中间形态
3. **完成展示**：成品飞机在画面中旋转展示
4. **属性面板**：展示该机型的五维属性雷达图

#### 步骤说明文字

每个折叠步骤配有简短的中文说明：

```
步骤 1：将纸张对折，打开后留下中线折痕
步骤 2：将左右两角沿中线向下折叠
步骤 3：将上方三角形沿虚线向下折叠
步骤 4：再次将左右两侧沿中线折叠
步骤 5：向外翻折两侧形成机翼
步骤 6：调整机翼角度，完成！
```

## 10. 后期扩展方向

以下内容为游戏 1.0 版本之后的扩展规划。

### 10.1 引入其他折纸模型

在主线通关后，引入纸飞机之外的折纸模型，丰富游戏玩法。

#### 纸鹤 — 空中支援型

| 项目 | 内容 |
|------|------|
| **定位** | 辅助单位，为纸飞机提供飞行中的 buff |
| **获取** | 通关后在学校向李老师学习折纸鹤 |
| **机制** | 赛前可额外装备一只纸鹤，飞行中纸鹤在飞机周围盘旋 |
| **效果** | 每 10 秒提供一次小型 buff（加速/回复耐久/提升稳定） |
| **升级** | 纸鹤可通过收集特殊材料升级，增强 buff 效果和频率 |
| **限制** | 只能装备一只，且会略微降低飞机 Speed（额外重量） |

#### 纸船 — 水面赛制

| 项目 | 内容 |
|------|------|
| **定位** | 全新赛道类型：水面赛道 |
| **获取** | 在全国赛后解锁的"海滨"大地图节点 |
| **机制** | 纸船在水面上漂流竞速，受水流和风浪影响 |
| **属性** | 新的五维属性：速度/稳定/水密性/操控/耐水性 |
| **特点** | 水面赛道有独特的障碍（漩涡、水草、石头）和天气（浪高、水流方向） |
| **奖励** | 水面赛道掉落水系专属零件，部分可跨类型使用 |

#### 纸忍者星 — 投掷干扰道具

| 项目 | 内容 |
|------|------|
| **定位** | 赛中使用的一次性干扰道具 |
| **获取** | 小卖部购买或比赛掉落 |
| **机制** | 飞行中投掷，命中对手造成减速/偏航/暂时失去技能 |
| **种类** | 普通纸星（减速 10%）、旋转纸星（偏航 15 度）、爆裂纸星（技能封印 3 秒） |
| **限制** | 每场比赛最多携带 2 个，投掷需要瞄准（技巧型操作） |
| **对策** | 护盾技能可挡住纸星，高 Stability 飞机受纸星影响更小 |

### 10.2 WebLLM 自定义工作台

利用 WebLLM 技术，为支持 WebGPU 的浏览器提供 AI 驱动的飞机自定义功能。

#### 功能概述

| 项目 | 内容 |
|------|------|
| **前置条件** | 浏览器支持 WebGPU（Chrome 113+、Edge 113+等） |
| **模型** | 本地运行的小型语言模型（通过 WebLLM 框架加载） |
| **首次加载** | 下载模型约 200-500MB，之后缓存在本地 |
| **可选功能** | 不影响核心游戏体验，纯粹的额外创意功能 |

#### 自定义流程

1. 玩家进入"自定义工作台"界面（大地图新节点：创意工坊）
2. 输入创意描述，例如：
   - "一架像鹰一样的纸飞机，在暴风中也能平稳飞行"
   - "闪电形状的超高速飞机，但是很脆弱"
   - "彩虹涂装的全能飞机"
3. AI 生成结果：
   - **飞机名称**：基于描述生成创意名称
   - **五维属性**：基于描述关键词分配属性（总属性点有上限限制）
   - **特殊效果描述**：生成一个被动效果的文字描述
   - **分类建议**：归类为速度型/特技型/稳定型
4. 玩家确认后，飞机进入**个人自定义池**

#### 平衡性控制

- AI 生成飞机的**总属性点不超过同级别标准飞机**
- 特殊效果从预定义效果池中匹配（AI 只决定匹配哪个，不创造新效果）
- 自定义飞机标记为"创意机型"，不影响排行榜和成就
- 每天最多生成 3 架自定义飞机

#### 降级方案

对于不支持 WebGPU 的浏览器：

- 提供基于模板的手动自定义功能
- 玩家从预设的名称/属性/效果列表中组合
- 功能简化但核心体验保留

### 10.3 社区功能

#### 飞机配置分享

- **导出码**：将飞机配置（机型+零件+技能）编码为短字符串
- **导入码**：输入其他玩家的导出码，在练习场试用该配置
- **配置展示**：查看导入配置的属性面板和五维雷达图
- **限制**：只能试用已解锁的零件和技能组合；未解锁的部分用默认替代

#### 种子码分享

- **赛事种子**：每次 Run 有一个唯一的 Seed 值
- **种子分享**：玩家可以将 Seed 分享给其他人
- **相同体验**：使用相同 Seed 会生成完全相同的锦标赛（赛道、天气、对手、随机事件）
- **竞速对比**：朋友之间可以比较相同 Seed 下的表现

#### 社区排行榜（本地存储 + 可选云同步）

| 排行榜 | 排名依据 | 说明 |
|--------|---------|------|
| 每日挑战榜 | 每日挑战得分 | 每日刷新 |
| 最高声望榜 | 声望总值 | 永久累计 |
| 速度纪录榜 | 单场最高速度 | 历史最高 |
| 图鉴完成榜 | 图鉴完成百分比 | 收集进度 |
| 难度挑战榜 | 最高修改器等级通关 | 挑战极限 |
